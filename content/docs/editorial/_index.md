問題のprefixに、難易度をつけている。

| 色  | 表記     |
| --- | -------- |
| 茶  | Br(own)  |
| 緑  | G(reen)  |
| 水  | C(yan)   |
| 青  | Bl(ue)   |
| 黄  | Y(ellow) |
| 橙  | O(range) |
| 赤  | R(ed)    |

各色の中で、だいたい三等分して、

- 下位なら`-`
- 中位なら無印
- 上位なら`+`

をつける。

例えば、Difficultyが1300なら、`(C-)ABC123D`とする。
